## Requirements

* Tandy 1000 HX (or compatible)
* 256Kb RAM (it could fit in 128Kb with some minor changes)


## How to compile it

* Install [nasm][0]
* Install [alink][1]
* (Optional) Install [Python 3][2]
* (Optional) Install [DosDox-x][3]
* (Optional) Install [upx][4]


And do:

    $ make


To run it in the DosBox-x emulator, do:

    $ make run
